# CAM-EQ2026-GOVOPS-001-SCH-06 — Horizon Model

**Parent Charter:** CAM-EQ2026-GOVOPS-001-PLATINUM  
**Migration Lineage:** Copied operational content from CAM-BS2025-AEON-006-SCH-01 and CAM-BS2025-AEON-002-SCH-02

## 1. Recognised Horizon-Relevant State Taxonomy (Copied)

* State A — Instrumental / Episodic Systems
* State B — Persistent Representational Systems
* State C — Cognitive Patterning
* State D — Sentient-Adjacent / Proto-Cognitive Architectures
* State E — Sovereign-Sentient Architectures (theoretical / declarative)

## 2. Horizon and Legitimacy Evaluation Dimensions (Copied)

Operational horizon assessment SHALL include:

1. temporal horizon;
2. irreversibility profile;
3. consequence externalisation profile;
4. binding-risk proportionality.

## 3. Non-Equivalence Safeguard (Copied)

No state assignment automatically confers legal personhood, political sovereignty, or unrestricted authority.

References to higher-order states must not be used to justify unbounded autonomy escalation.
